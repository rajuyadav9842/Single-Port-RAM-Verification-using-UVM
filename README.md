# SPRAM_UVM_Verification

This repository contains the RTL design and UVM-based verification of Single Port RAM (SPRAM) using SystemVerilog.
The project includes read/write operation verification, functional coverage, and a complete UVM testbench environment.

## Features
- SPRAM RTL Design
- UVM-Based Verification
- Read/Write Transactions
- Driver, Monitor, Scoreboard, Agent, Environment
- Functional Coverage
- Waveform Simulation

## Tools
- Verilog/SystemVerilog
- UVM
- QuestaSim
- Cadence Xcelium
